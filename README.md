## Projetos Principais

### [`prompt-generator`](https://github.com/REDSEC90/prompt-generator)

Ferramenta de linha de comando para criar, revisar, executar e melhorar prompts estruturados para modelos de linguagem.

| Área | Detalhes |
|---|---|
| Tecnologias | TypeScript, Node.js, Jest, Ollama, Anthropic, OpenAI |
| Foco | Engenharia de prompts, fluxos com modelos de linguagem, feedback, conjuntos de dados, avaliação e treinamento local |
| Sinal técnico | Mostra uso prático de IA, automação, organização de CLI e documentação de ferramenta real |

**Destaques**

- Modos de execução local, remoto e offline.
- Integração com Ollama, Anthropic e OpenAI.
- Armazenamento de feedback para melhoria contínua.
- Exportação de conjuntos de dados.
- Fluxo de avaliação e treinamento.
- Estrutura com testes, documentação e arquitetura reutilizável.

---

### [`wdroid-v2`](https://github.com/REDSEC90/wdroid-v2)

Ambiente modular em linha de comando para controlar e automatizar instâncias Android/Waydroid no Linux.

| Área | Detalhes |
|---|---|
| Tecnologias | Bash/Shell, Linux, Debian, Waydroid, ADB, Makefile |
| Foco | Automação Android, controle de estado, diagnóstico, logs, backups e plugins |
| Sinal técnico | Mostra pensamento de engenharia de sistemas aplicado a automação real no Linux |

**Destaques**

- Controle de ciclo de vida por máquina de estados.
- Logs estruturados, lockfile, tentativas automáticas e diagnósticos.
- Integração com ADB, Waydroid, backups e sistema de plugins.
- Organização modular em vez de scripts soltos.
- Base útil para automação, testes locais e controle de ambientes Android.
