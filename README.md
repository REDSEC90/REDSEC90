## Flagship Work

### [`prompt-generator`](https://github.com/REDSEC90/prompt-generator)

CLI for creating, reviewing, running and improving structured prompts for LLMs.

| Area | Details |
|---|---|
| Stack | TypeScript, Node.js, Jest, Ollama, Anthropic, OpenAI |
| Focus | Prompt engineering, LLM workflows, feedback, datasets, evaluation and local fine-tuning |
| Why it matters | Shows practical AI tooling, not only AI usage |

**Highlights**

- Local, remote and offline execution modes.
- Integrations with Ollama, Anthropic and OpenAI.
- Feedback storage and continuous improvement loop.
- Dataset export, training flow and evaluation structure.
- Tests and documentation around a real CLI workflow.

---

### [`wdroid-v2`](https://github.com/REDSEC90/wdroid-v2)

Modular Linux CLI runtime for controlling Android/Waydroid environments.

| Area | Details |
|---|---|
| Stack | Bash/Shell, Linux, Debian, Waydroid, ADB, Makefile |
| Focus | Android automation, state control, diagnostics, logging, backups and plugins |
| Why it matters | Shows systems thinking applied to real Linux automation |

**Highlights**

- Lifecycle control through a state machine.
- Structured logs, lockfile, retry and diagnostic commands.
- Integration with ADB, Waydroid, backups and plugin execution.
- Modular shell architecture instead of one-off scripts.
